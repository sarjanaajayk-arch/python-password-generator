# Python Password Generator

A secure password generator built with Python using random and string modules to generate strong passwords.

## Features
- Generates random passwords
- Supports uppercase and lowercase letters
- Includes numbers and special symbols
- User-defined password length
- Input validation for password length

## Technologies Used
- Python
- random module
- string module

## How to Run
1. Open terminal
2. Run the Python file
3. Enter password length
4. Get generated password

## Sample Output
